# Makao Realtors

Elevating Kenyan Living — Premium real estate locator and listing platform in Kenya.

Makao Realtors is a gateway to premium living in Kenya, bridging the gap between discerning residents and high-quality spaces through technology, verification, and transparency.

## Core Features

- **Premium Property Locator**: Discover exclusive properties across all 47 Kenyan counties.
- **Verified Listings**: Every property undergoes a rigorous 3-step authentication protocol.
- **Neighborhood Insights**: Gain deep context on elite residential areas like Karen, Runda, and Kilimani.
- **Advanced Search**: Robust search capabilities by title, neighborhood, or location.
- **Digital Ecosystem**: Virtual tours, secure interactions, and architectural inspiration.

## Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Database/Auth**: [Supabase](https://supabase.com/)

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Development

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the application for production.
- `npm run lint`: Runs TypeScript checks.
